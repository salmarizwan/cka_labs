# Core Concepts Labs

This folder contains hands-on exercises covering the **Core Concepts domain** of the CKA exam.

## Topics Covered
- Pods: creation, deletion, multi-container examples
- Namespaces: dev/prod isolation, resource scoping
- Labels & Annotations: selection, filtering, organization

## Lab Format
- YAML manifests included
- CLI commands with expected outputs
- Exam-focused tips for each scenario
